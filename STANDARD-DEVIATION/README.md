## 2. Introduction

In statistics, standard deviation is a fundamental measure of dispersion that quantifies the extent to which data values deviate from the mean. When multiple datasets share the same mean, the dataset with the smallest standard deviation is considered more consistent, as its data points are more closely clustered around the mean.

This analysis evaluates three datasets with identical means but different standard deviations to determine which dataset is statistically preferable.
